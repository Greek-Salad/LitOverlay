# 📖 LitOverlay

Современная платформа для чтения интерактивных книг с поддержкой мультимедиа, интеллектуальной системой подсказок и глубокой кастомизацией интерфейса.

## 📚 Читать мои произведения:

👉 [**Открыть в браузере**](https://greek-salad.github.io/LitOverlay/)

## 🌟 Ключевые возможности

### 📚 Мультимедийная читалка
- **Динамическая загрузка глав** — книги разбиты на отдельные HTML-файлы, подгружаются по мере чтения
- **Интеллектуальная вставка медиа** — аудио и изображения автоматически встраиваются в текст по заданным правилам
- **Интерактивные подсказки (hints)** — всплывающие пояснения к терминам, сноски, переводы
- **Подсветка абзацев** — клик по абзацу выделяет его (удобно для стримеров и публичных чтений)
- **Полнотекстовый поиск** — с навигацией по результатам и подсветкой вхождений

### 🎨 Гибкая настройка внешнего вида
- **Светлая/тёмная темы** — быстрое переключение пресетов
- **Кастомный выбор цветов** — собственный виджет для точной настройки фона и текста
- **Библиотека шрифтов** — Lato, Antiqua, Liberation Sans/Serif, Roboto, Source Code Pro, Whitney
- **Регулировка** — размера текста, ширины области чтения, межстрочного интервала

### 🎵 Централизованное аудио
- **Единый аудиоменеджер** — синхронизация воспроизведения по всему приложению
- **Глобальный плеер в тулбаре** — управление текущим треком из любой точки
- **Автоматическая остановка** — при переходе между главами музыка ставится на паузу
- **Отображение текста песни** — отдельная панель с лирикой для аудиофайлов

### 📋 Библиотека книг
- **Автоматическое сканирование** — книги подгружаются из структурированного каталога
- **Метаданные** — обложка, автор, дата написания, теги, возрастной рейтинг
- **Возрастной гейт** — для книг 18+ требуется подтверждение возраста (с сессионным хранением)
- **Индикация статуса** — завершённые и находящиеся в процессе произведения
- **Фильтрация по тегам** — отображение жанров и особенностей книги

### 📱 Продуманный UX
- **Индикатор прогресса чтения** — показывает % прочитанного в текущей главе
- **Оглавление** — быстрая навигация по главам
- **Адаптивный дизайн** — комфортное чтение на любых устройствах
- **Сохранение настроек** — тема, шрифты и последняя глава хранятся в localStorage
- **PWA-ready** — манифест для установки на устройство

## 🚀 Быстрый старт

### Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Greek-Salad/LitOverlay.git
   cd LitOverlay
   ```

2. **Запустите локальный сервер** (обязательно, так как используются fetch-запросы):
   - **VS Code:** установите расширение Live Server и нажмите "Go Live"
   - **Node.js:** ```bash npx http-server```
   - **Python:** ```bash python -m http.server 8000```

3. **Откройте в браузере:** `http://localhost:8000/index.html`

### Добавление новой книги

1. Создайте папку с уникальным ID в `books/` (например, `books/my-book/`)
2. Добавьте `info.json`:
   ```json
   {
     "id": "my-book",
     "title": "Название книги",
     "author": "Имя автора",
     "writtenDate": "2024-01-01",
     "description": "Описание...",
     "tags": ["фантастика", "приключения"],
     "cover": "cover.png",
     "finished": true,
     "totalChapters": 10,
     "hasMedia": true,
     "hasHints": true,
     "ageRating": 16
   }
   ```
3. Поместите файлы глав в `chapters/` с именами `01.html`, `02.html`...
4. (Опционально) Добавьте `media-rules.json` и `hint-rules.json` для специфичных для книги правил
5. Добавьте ID книги в `books/index.json`

### Формат главы

```html
<h2>Название главы</h2>
<p>Текст первого абзаца...</p>
<p>Текст второго абзаца...</p>
```

## 🎯 Конфигурация

### Правила для медиа (`media-rules.json`)

Аудио:
```json
{
  "id": "audio-1",
  "type": "audio",
  "src": ["track.mp3"],
  "title": "Исполнитель — Название трека",
  "chapter": 3,
  "position": "after",  // "before" или "after"
  "anchor": "Точный текст абзаца для привязки"
}
```

Изображения:
```json
{
  "id": "image-1",
  "type": "image",
  "src": ["image1.png", "image2.png"],
  "chapter": 5,
  "position": "before",
  "anchor": "Текст для привязки",
  "width": "80%",
  "height": "auto",
  "alt": "Описание"
}
```

### Правила для подсказок (`hint-rules.json`)

```json
{
  "hints": [
    {
      "chapter": 2,
      "text": "слово или фраза",
      "hint": "Текст подсказки, которая появится при наведении"
    }
  ]
}
```

## 🖥️ Использование

### В читалке
- **📖 Навигация:** боковое меню (кнопка ≡) или кнопки "Начало"/"Следующая глава"
- **🎨 Тема:** иконка луны/солнца в тулбаре для переключения пресетов и ручной настройки цветов
- **⚙️ Настройки:** иконка шестерёнки для шрифтов, размера, ширины
- **🔍 Поиск:** поле поиска в центре тулбара (Enter — следующее, Shift+Enter — предыдущее)
- **🎵 Аудио:** при появлении аудиоплеера в тексте, иконка ноты в тулбаре открывает глобальный плеер
- **💡 Подсказки:** наведите на подчёркнутое слово, чтобы увидеть пояснение
- **✏️ Подсветка:** клик по абзацу выделяет его (повторный клик снимает выделение)

### В библиотеке
- Клик по карточке книги открывает читалку
- Книги 18+ требуют подтверждения возраста
- Статус ✅ Завершено / 🔄 В процессе отображается в метаданных

## 📦 Технические требования

- Современный браузер с поддержкой:
  - ES6+ (модули, классы, async/await)
  - Fetch API
  - localStorage
  - CSS Custom Properties
  - Flexbox/Grid
  - WOFF2 шрифты
